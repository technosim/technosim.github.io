## Networking

- **[Cable Academy - trueCABLE](https://www.truecable.com/blogs/cable-academy)**
  <br><span class="text-muted small">A great resource to learn about network cabling and related topics.</span>

- **[BGP Toolkit](https://bgp.he.net)**
  <br><span class="text-muted small">Explore BGP routing, AS paths, and internet topology.</span>

- **[Netmiko Library](https://pynet.twb-tech.com/blog/netmiko-python-library.html)**
  <br><span class="text-muted small">Open-Source Python Library for Network Automation.</span>

- **[DNS Propagation Checker](https://www.whatsmydns.net/#A/)**
  <br><span class="text-muted small">Check the status of a DNS record against multiple name servers across the globe.</span>

- **[Network Administration Tools & Software](https://www.netadmintools.com/)**
  <br><span class="text-muted small">Categorical lists and reviews of varying network administration tools.</span>

- **[GitHub - yuriskinfo / cheat-sheets](https://github.com/yuriskinfo/cheat-sheets/tree/master)**
  <br><span class="text-muted small">Configuration, Debug and Diagnostics cheat sheets for Network and Linux based equipment. Other repos here are also very useful.</span>

- **[DNSCHECKER - Port Scanner](https://dnschecker.org/port-scanner.php?)**
  <br><span class="text-muted small">Check the availability of ports associated with a specific domain or IP address.</span>

- **[IANA - Service Name and Transport Protocol Port Number Registry](https://www.iana.org/assignments/service-names-port-numbers/service-names-port-numbers.xhtml?&page=1)**
  <br><span class="text-muted small">Check the official IANA registry for service and port numbers.</span>

## Security

- **[FireHOL IP Lists](https://iplists.firehol.org/)**
  <br><span class="text-muted small">Open source threat feeds.</span>

- **[NIST Technical Guide to Information Security Testing and Assessment](https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-115.pdf)**
  <br><span class="text-muted small">A good basis for pentesting.</span>

- **[U.S. DISA Security Technical Implementation Guides (DISA STIGs)](https://stigviewer.cyberprotection.com/stigs)**
  <br><span class="text-muted small">Security hardening standards for IT systems.</span>

- **[Center for Internet Security (CIS) Benchmarks](https://www.cisecurity.org/cis-benchmarks)**
  <br><span class="text-muted small">Also security hardening standards for IT systems.</span>

- **[threatfeeds.io](https://threatfeeds.io/)**
  <br><span class="text-muted small">Free and open-source threat intelligence feeds.</span>

- **[urlscan.io](https://urlscan.io/)**
  <br><span class="text-muted small">Check an untrusted website in a sandboxed environment.</span>
